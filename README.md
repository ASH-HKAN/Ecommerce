# Auto Tools E-Commerce Platform

This repository is planned as a professional and enterprise-ready e-commerce platform for selling auto tools. The system includes a customer storefront, user panel, admin panel, basket/cart, reservation system, payments, inventory management, CRM features, and Nginx-based deployment.

## Recommended Architecture

```text
Next.js frontend + NestJS backend + PostgreSQL + Prisma + Redis + Stripe + Nginx + Docker
```
